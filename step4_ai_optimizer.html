the core learning rule

At each step the agent computes how the error changes with spacing d, then nudges d in the downhill direction — identical to how neural networks learn weights.

Loss Function
MSE
Mean Squared Error on λ

L = (λ_pred − λ_target)². Minimising this is equivalent to finding the exact nanostructure geometry that produces your target structural color.
